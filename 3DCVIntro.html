Welcome to the Interactive 3D CV

This is an Experimental 3D Interactive CV Experience. It is made using three.js and many useful extensions.

The webpage and text boxes are true rendered HTML elements within 3D space. Some browsers will sometimes let you interact within the iframe. That's pretty cool when that happens.

Controls:

Walk Forward - W
Walk Back - S
Rotate Left - A
Rotate Right - D

Some objects may be clicked on
Performance may vary - desktop system using Chrome or Safari recommended.
